# VideoWebSite

基于 Node.js + Express 的本地视频文件浏览与播放 Web 应用，暗色主题，支持 PC / 移动端自适应。

---

## 现有功能

### 视频管理
- **递归扫描**：自动扫描项目目录及子目录，支持 `.mp4` `.mkv` `.avi` `.mov` `.flv` `.wmv` `.webm` `.ogg`
- **搜索过滤**：按文件名实时搜索
- **智能排序**：按文件名自然排序（支持数字序号）

### 播放控制
- **Play / Pause**：合并为单一按钮，点击视频画面也可切换
- **进度条**：可拖拽/点击定位，支持触摸滑动
- **音量控制**：滑块调节 + 一键静音，偏好自动保存
- **倍速播放**：0.5x ~ 2x，偏好自动保存
- **时间跳转**：±10s（快捷按钮)、±30s / ±1min / ±5min（扩展栏）
- **画中画 (PiP)**：支持悬浮小窗播放
- **全屏**：按钮或双击视频画面切换

### 响应式设计
- **PC 端**：左侧固定侧边栏 + 右侧播放区域
- **移动端**：侧边栏变为底部抽屉，上滑展开 / 下滑收起，点击遮罩关闭
- **触摸手势**：左右滑动快进/快退 10s
- 全触屏操作友好，所有交互按钮 44px+ 适配手指点击

### 键盘快捷键

| 按键 | 功能 |
|---|---|
| `Space` / `K` | 播放 / 暂停 |
| `←` / `→` | 快退 / 快进 10s |
| `Shift + ←/→` | 快退 / 快进 1min |
| `Ctrl + ←/→` | 快退 / 快进 5s |
| `↑` / `↓` | 音量 ±5% |
| `M` | 静音切换 |
| `F` | 全屏切换 |
| `T` | 侧边栏切换 |
| `1` ~ `9` | 直接播放对应序号的视频 |
| `Esc` | 关闭移动端侧边栏 / 退出全屏 |

### 偏好记忆
- 音量、播放速度通过 `localStorage` 自动记忆，刷新页面后恢复

## 项目结构

```
VideoWebSite/
├── server.js          # Express 后端 —— 视频扫描 / API / 静态服务
├── package.json       # 项目配置与依赖
├── public/
│   ├── index.html     # 前端主页面（语义化结构，SVG 图标）
│   ├── scripts.js     # 前端逻辑（模块化 IIFE，完备事件处理）
│   └── styles.css     # 暗色主题样式（CSS 变量，响应式）
└── node_modules/      # 依赖
```

## 技术栈

| 层 | 技术 |
|---|---|
| 运行时 | Node.js |
| 后端框架 | Express 4.x |
| 前端 | 原生 HTML5 / CSS3 / JS (ES6+) |
| 视频播放 | HTML5 `<video>` |
| 图标 | 内联 SVG（零外部依赖） |

## 安装与使用

### 环境要求
- Node.js（推荐 18+ LTS）

### 启动

```bash
npm install
# 放置视频文件到项目目录（或任意子目录）
npm start
```

访问 `http://localhost:4007`。

### 自定义端口

```bash
PORT=8080 npm start    # Linux / macOS
```
```powershell
$env:PORT=8080; npm start   # Windows PowerShell
```
